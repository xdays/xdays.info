# xdays.me [![Build Status](https://travis-ci.org/xdays/xdays.me.svg?branch=travis)](https://travis-ci.org/xdays/xdays.me)

我的博客使用pelican静态博客引擎，本仓库托管所有博文的markdown文件。

# CI
* 用Git管理所有博客的变动
* 用Github的webhook来出发博客渲染请求
* 用Nginx+Lua处理webhook请求渲染博客
